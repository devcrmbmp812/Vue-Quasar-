# Yoga IgniteSystem project

## Getting started

```bash
# 1. Install Quasar CLI
# install vue-cli globally
$ npm install -g vue-cli # or @vue/cli @vue/cli-init // recommended
# or:
$ yarn global add vue-cli # or @vue/cli @vue/cli-init

# Node.js >= 8.9.0 is required.
$ yarn global add quasar-cli
# or:
$ npm install -g quasar-cli

# 2. Clone the repository.
git clone git@bitbucket.org:ignitesystems/yoga_cj2.git

# 3. Enter your newly-cloned folder
cd yoga_cj2

# 4. Install dependencies. Make sure yarn is installed: https://yarnpkg.com/lang/en/docs/install
yarn

# 5. Launch the dev server
quasar dev

# 6. Build App for Production
quasar build

# 7. Serve Static-Content Folder
#This command can be used in production too.
quasar serve [path]

#for example:
quasar serve dist/spa-mat
```
